Bem-vindo à documentação do 3PST_2.0.0!

3PST 2.0.0 – Sistema web para gestão do portfólio de Programas, Projetos, Produtos, Serviços e Tecnologias do INPE.

Partes interessadas

  • Funcionários do SEPEC: funcionários do setor, que participam diretamente da gestão dos dados da carteira;

  • Colaboradores da COGPI: colaboradores da coordenação, que participam indiretamente na gestão dos dados da carteira;

  • Funcionários do INPE: funcionários da instituição que pretendem abrir TAPs, TAPgs e TAS.

Documentos

habilitarInput(elemenId)

A função habilitarInput() habilita todos os campos de entrada de um formulário e desabilita o botão “Alterar” ao ativar o botão “Salvar”.

Argumentos:

  • elemenId (string) – Retorna uma referência ao primeiro objeto com o valor especificado do atributo ID.

downloadCSVFile(csv, filename, tagname)

A função downloadCSVFile é responsável por criar um arquivo CSV e iniciar seu download.

Argumentos:

  • csv (string) – O parâmetro csv é uma string que representa o conteúdo do arquivo CSV que você deseja baixar.

  • filename (string) – O parâmetro ‘filename’ é uma string que especifica o nome do arquivo a ser baixado.

  • tagname (string) – Cria uma instância do elemento para a tag especificada.

htmlToCsv(filename)

A função htmlToCsv converte uma tabela HTML em um arquivo CSV e faz o download.

Argumentos:

  • filename (string) – O parâmetro filename é uma string que representa o nome do arquivo CSV que será baixado.
